# Diagram Creator Skill

Diagram Creator 是一款通过 AI 图像生成，将你的想法直接转化为专业信息图表级别图表的技能。只需用自然语言描述你想要的内容——流程图、架构图、矩阵图或时间轴——AI 就会按照经过验证的视觉配方组装结构化 Prompt，在几秒内生成一张精美的图表。

## ✨ 核心能力

- **9 种图表类型，一个技能搞定**：无论你需要决策流程图、分层架构图、2×2 优先级矩阵、产品路线图时间轴，还是增长飞轮——只需用自然语言描述你的意图，AI 会自动匹配到正确的图表结构。
- **3 套精心策划的视觉风格**：每张图表都可以用 **Business**（麦肯锡/BCG 级咨询风格）、**Editorial Luxury**（暖色纸质纹理配衬线体排版）或 **Sketch**（手绘白板工作坊质感）风格渲染。AI 会自动选择默认风格，你也可以指定偏好。
- **明暗双模式**：每种风格都支持浅色和深色两种颜色模式。当你主动要求或嵌入到深色主题的幻灯片中时，深色模式会自动激活。
- **告别"标题污染"**：受够了 AI 生成的图表看起来像一整页 PPT、带着巨大标题的效果？内置的"Canvas Behavior"约束确保输出的是一张干净的、紧凑裁切的纯净图表——可以直接嵌入到任何地方。
- **即嵌即用**：图表既可以作为独立图片生成，也可以作为组件直接嵌入幻灯片、文档或网页中。嵌入时，你只需指定目标像素尺寸，技能会确保宽高比精确匹配，无裁切、无拉伸、无留白。

## 📐 支持的图表类型

| 类型 | 适用场景 | 示例 |
|:---|:---|:---|
| **Quadrant（象限图）** | 优先级分析、SWOT、竞品映射 | 按两个维度定位项目的 2×2 矩阵 |
| **Timeline（时间轴）** | 路线图、里程碑、历史演进 | 带有事件标记的横向/纵向时间线 |
| **Flowchart（流程图）** | 工作流程、决策树、审批流程 | 带分支和循环的逐步流程 |
| **Architecture（架构图）** | 系统设计、技术栈、微服务 | 分层模块及其连接关系 |
| **Pyramid（金字塔）** | 层级模型、马斯洛需求、成熟度框架 | 自底向上的堆叠层级 |
| **Cycle（循环图）** | 迭代流程、PDCA、反馈回路 | 连续循环的环形图 |
| **Funnel（漏斗图）** | 转化漏斗、销售阶段、筛选过程 | 从上宽到下窄的阶段 |
| **Org Chart（组织架构图）** | 团队结构、汇报关系、层级体系 | 带父子节点的树状图 |
| **Flywheel（飞轮图）** | 增长引擎、自增强循环 | 环形动量图 |

## 🎨 视觉风格

### 1. Business（商务经典）
> 麦肯锡/BCG 级咨询图表品质。纯白背景上的深海军蓝点缀，锐利几何形状，Helvetica 字体，零装饰噪音。完美适配董事会演示与战略汇报。

### 2. Editorial Luxury（暖调奢华）
> 温暖的象牙色纸底搭配青铜色点缀，优雅的衬线体排版（Playfair Display），极细分割线，充裕的呼吸留白。非常适合年度报告、品牌叙事和高层讲述。

### 3. Sketch（手绘白板）
> 真实的速记笔记美学：手绘的不规则边框、硬偏移阴影、圆点网格纸质纹理、马克笔风格的标题字体以及便利贴黄色的强调色块。特别适合头脑风暴、工作坊和创意提案。

每种风格还支持**深色模式 (Dark Mode)**——反转背景明暗，同时保留该风格的独特特征（线条粗细、节点形状、装饰元素保持不变）。

## 🚀 安装与使用方法

### 方式一：技能安装器

如果你的 AI 环境支持指令式安装：

```bash
npx skills add exceedhl/diagram-creator-skill
```

### 方式二：直接复制目录

1. 下载或 Clone 本项目。
2. 将整个 `diagram-creator-skill` 文件夹复制到你工程空间里的 Agent 技能目录（例如 `.agents/skills/` 之下）。
3. Agent 会在上下文中自动识别到 `SKILL.md` 的存在并自主加载图表生成能力。

## 💬 Prompt 触发示例

**示例一：快速生成流程图**
> "生成一张流程图，展示我们的 CI/CD 流水线：代码推送 → 构建 → 测试 → 部署到预发环境 → 审批 → 生产发布。使用 Sketch 风格。"

**示例二：战略矩阵**
> "创建一个 2×2 优先级矩阵。X 轴是实施难度（低到高），Y 轴是商业影响（低到高）。标注以下项目：AI 客服（高影响、低难度），ERP 迁移（高影响、高难度），办公室改造（低影响、高难度），邮件模板（低影响、低难度）。使用 Business 风格。"

**示例三：架构图**
> "画一个三层系统架构：数据层（PostgreSQL, Redis, S3），服务层（Auth, API Gateway, ML Pipeline），客户端层（Web, Mobile, CLI）。展示层间连接关系。Editorial Luxury 风格，深色模式。"

**示例四：增长飞轮**
> "创建一个 4 阶段的飞轮图：更好的产品 → 更多用户 → 更多数据 → 更强的 AI 模型 →（回到更好的产品）。Sketch 风格。"
